# homebrew-tap

Homebrew tap for [ai-usagebar](https://github.com/Zenardi/ai-usagebar) — a status-bar widget and TUI for AI plan usage.

```bash
brew install Zenardi/tap/ai-usagebar
```

Apple Silicon only (CI publishes an arm64 binary). Intel Macs build from source — see the [main repo](https://github.com/Zenardi/ai-usagebar#macos-sketchybar).
